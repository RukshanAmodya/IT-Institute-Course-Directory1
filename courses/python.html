Course Overview

This course covers Python Programming from scratch up to intermediate/advanced concepts. Students will learn procedural scripting, Object-Oriented programming in Python, file manipulation, data structures, and get introduced to scientific libraries like NumPy and Pandas, culminating in hands-on backend script development.

Comprehensive Curriculum Structure

The syllabus is structured into 3 main modules, each consisting of specific chapters and practical sub-topics:

  1. Module 1: Python Basics & Scripting
    1. Introduction to Python and Setup
      1. Python Interpreter, virtual environments, and package managers (pip)
      2. Basic types: integers, floats, booleans, strings
      3. Input/Output functions (input(), print() formatting)
    2. Control Flow and Data Structures
      1. Conditionals (if, elif, else)
      2. Loops (while, for, nested iterations)
      3. Native collections: Lists, Tuples, Dictionaries, and Sets
    3. Functions and Modules
      1. Defining functions, positional/keyword arguments, return statements
      2. Scope of variables (global vs. local)
      3. Importing built-in modules (math, random, datetime)
  2. Module 2: Object-Oriented & Advanced Python
    1. OOP Paradigms in Python
      1. Classes, instance variables, methods, and __init__ constructor
      2. Inheritance and polymorphism in Python
      3. Special dunder/magic methods (__str__, __repr__, __len__)
    2. Advanced Python Features
      1. List comprehensions and generator expressions
      2. Decorators, closures, and generator functions
      3. File manipulation (reading/writing CSV, JSON, and text files)
  3. Module 3: Data Fundamentals & Web Intro
    1. Scientific Computing & Data Manipulation
      1. Introduction to NumPy arrays and vectorized operations
      2. Introduction to Pandas Series and DataFrames
      3. Data cleaning, filtering, and grouping techniques
    2. Basic Web Scraping & APIs
      1. Making HTTP requests using requests library
      2. Parsing HTML structures using BeautifulSoup
      3. Interacting with public JSON REST APIs
