Course Overview

This intensive full-stack program equips students with modern frontend and backend development skills. Starting with structural HTML5 and CSS3 layouts, students transition into vanilla JavaScript DOM manipulation, asynchronous programming, server-side engineering with Node.js, API development, and SQL databases.

Comprehensive Curriculum Structure

The syllabus is structured into 3 main modules, each consisting of specific chapters and practical sub-topics:

  1. Module 1: Front-End Core Foundations
    1. Semantic HTML5 & Accessibility
      1. HTML document structure, meta tags, and browser rendering lifecycle
      2. Semantic markup: article, section, header, footer, nav, aside
      3. Accessible forms, ARIA attributes, and semantic table structures
    2. Styling with Modern CSS
      1. CSS Box Model: margins, borders, padding, sizing
      2. Layout systems: CSS Flexbox and CSS Grid
      3. Responsive web design: Media queries and viewport configurations
  2. Module 2: JavaScript & Client-Side Logic
    1. JavaScript Core Syntax
      1. Variables (let, const), data types, scope, and template literals
      2. Functions, arrow functions, higher-order functions
      3. Arrays, Objects, destructuring, and spread/rest operators
    2. DOM Manipulation & Asynchronous Control
      1. Selecting elements, modifying content, classes, attributes
      2. Handling events, event propagation (bubbling vs. capturing)
      3. Asynchronous JS: Callbacks, Promises, and Async/Await with Fetch API
  3. Module 3: Back-End & Database Engineering
    1. Server-Side Development with Node.js & Express
      1. Node.js runtime environment, event loop, and file system module
      2. Setting up Express.js application and routing
      3. RESTful API design and middleware implementation
    2. Databases & Authentication
      1. Relational database modeling with SQL and PostgreSQL
      2. Querying tables, joins, transactions, and indexing
      3. JSON Web Token (JWT) session-less authentication
